# 리포트와 화면 안내

이 문서는 "이 화면이 대체 뭘 보는 곳인지"를 빠르게 이해하기 위한 안내서입니다.

## 먼저 이렇게 보면 됩니다

- `오늘`: 처음 들어오는 시작 화면입니다. 오늘 기준일, 경고, 주목 종목, 최신 보고서를 먼저 봅니다.
- `시장 현황`: 오늘 장이 강한지 약한지, 시장 뉴스가 어떤 분위기인지 보는 화면입니다.
- `리더보드`: 지금 눈여겨볼 종목 후보를 보는 화면입니다.
- `포트폴리오`: 후보를 실제 보유안으로 바꾸면 어떤 모습이 되는지 보는 화면입니다.
- `사후 평가`: 시간이 지난 뒤 추천이 실제로 어땠는지 점검하는 화면입니다.
- `장중 콘솔`: 장중에 판단이 어떻게 바뀌었는지 연구용으로 보는 화면입니다.
- `운영` / `헬스 대시보드`: 시스템이 정상적으로 돌아가는지 확인하는 운영 화면입니다.
- `리서치 랩`: 일반 확인용이 아니라 고급 연구와 수동 반영 비교용 화면입니다.

## 기준 리포트

- `daily_research_report`: 오늘 시장과 주목 종목을 읽기 쉽게 정리한 요약
- `portfolio_report`: 목표 보유안과 리밸런스 방향을 정리한 요약
- `evaluation_report`: 시간이 지난 뒤 추천 결과가 어땠는지 보는 사후 평가 요약
- `intraday_summary_report`: 장중 판단 흐름을 시간대별로 요약한 보고서
- `evaluation_postmortem_report`: 추천 결과를 다시 점검하는 사후 점검 보고서
- `daily_discord_preview`: 디스코드로 보내는 장마감 요약
- `release_candidate_checklist`: 운영 점검표

## 화면별 의미

- `오늘`
  - 어떤 화면인가: 앱을 열자마자 가장 먼저 보는 시작 화면
  - 뭘 먼저 보나: 기준일, 경고, 오늘 주목 종목, 최신 보고서

- `시장 현황`
  - 어떤 화면인가: 시장 전체 분위기를 먼저 읽는 화면
  - 뭘 먼저 보나: 시장 흐름, 상승 비율, 수급, 시장 뉴스

- `리더보드`
  - 어떤 화면인가: 현재 주목할 종목 순위를 보는 화면
  - 뭘 먼저 보나: 등급, 종합점수, 참고 기대수익, 주의 신호

- `포트폴리오`
  - 어떤 화면인가: 추천 종목을 실제 보유안으로 바꾸는 화면
  - 뭘 먼저 보나: 목표 보유안, 리밸런스 계획, 대기/차단 사유

- `포트폴리오 평가`
  - 어떤 화면인가: 포트폴리오 제안이 시간이 지나며 어땠는지 보는 화면
  - 뭘 먼저 보나: 순자산 가치 추이, 방식별 비교 요약

- `장중 콘솔`
  - 어떤 화면인가: 장중에 원래 판단이 보정되고 최종 판단으로 바뀌는 과정을 보는 연구용 화면
  - 뭘 먼저 보나: 세션 요약, 후보 종목, 최종 판단, 시점 결과

- `사후 평가`
  - 어떤 화면인가: 1거래일, 5거래일 뒤 결과를 점검하는 화면
  - 뭘 먼저 보나: 최신 평가 요약, 평가 결과 샘플, 예상 범위 점검

- `종목 분석`
  - 어떤 화면인가: 한 종목만 깊게 보는 화면
  - 뭘 먼저 보나: 종목 요약, 가격/수급, 장중 판단, 사후 결과

- `리서치 랩`
  - 어떤 화면인가: 정책/메타 모델 실험과 수동 반영 후보를 비교하는 고급 화면
  - 뭘 먼저 보나: 기능 상태, 정책 대비 메타 오버레이, 수동 반영 전 비교표

- `운영`
  - 어떤 화면인가: 최근 배치, 실패, 알림, 스케줄러 상태를 점검하는 운영자 화면
  - 뭘 먼저 보나: 운영 요약, 스케줄러 상태, 최근 실행 이력

- `헬스 대시보드`
  - 어떤 화면인가: 시스템 건강 상태를 한 화면에 모아 보는 점검판
  - 뭘 먼저 보나: 전체 헬스 요약, 활성 락, 복구 대기열, 경고

- `문서 / 도움말`
  - 어떤 화면인가: 용어, 사용법, 운영 문서를 찾는 화면
  - 뭘 먼저 보나: 사용자 가이드, 화면 안내, 서버 운영 문서
